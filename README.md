## 목적
각 데이터는 주어져 있으며, 언제 공장을 가동하고 멈출지, 그리고 검사를 할지를 판단하는 모델을 학습하는 것이 주목적입니다.
- 정해진 수요에 맞춰 최적 블럭 장난감 생산 공정 설계
- AI 기반 알고리즘으로 공정 계획을 만들어 csv 파일 제출
따라서, 유전 알고리즘과 DQN을 기반으로 한 모델을 각각 만들고, 두 모델에 대한 학습을 진행했습니다.
